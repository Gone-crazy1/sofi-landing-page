# Sofi AI Wallet Landing Page

A modern, responsive landing page for Sofi AI Mini Wallet - empowering everyday Nigerian businesses with digital financial solutions.

## 🎯 Project Overview

This landing page is designed to attract and convert Nigerian small business owners including:
- Market women (Iya Alata) selling pepper and tomatoes
- Meat sellers (Butchers)
- POS agents
- Pharmacists (Chemist shops)
- Filling station attendants/owners
- Electronics sellers
- Keke/Okada riders
- Clothing and boutique owners
- Car spare parts sellers
- Barbers and hairdressers
- Small café/food joint owners (Buka)

## ✨ Features

### 🎨 Design Features
- **Modern, Clean Design**: Beautiful gradient backgrounds and smooth animations
- **Mobile-First Responsive**: Optimized for all devices, especially mobile
- **Fast Loading**: Optimized for Nigerian internet speeds
- **Smooth Scrolling**: Seamless navigation experience
- **Interactive Elements**: Hover effects, animations, and micro-interactions

### 💼 Business Features Highlighted
- Virtual Account Creation (Powered by Monnify/Paystack)
- Airtime & Data Purchase
- Secure Transfers to Any Bank
- Crypto Deposit (USDT/BTC accepted)
- POS Services for Agents
- Instant Alerts and Receipts
- Zero BVN Onboarding
- Bank-Level Security

### 🏦 Trust Indicators
- **Trusted By Section**: Displays logos of major Nigerian banks and fintech companies
- **Social Proof**: Real-life photo carousel of target users
- **Statistics**: User count, transaction volume, and support availability

## 🚀 Technical Stack

- **HTML5**: Semantic, accessible markup
- **CSS3**: Modern styling with Flexbox and Grid
- **Vanilla JavaScript**: No dependencies, fast loading
- **Google Fonts**: Inter font family for clean typography
- **Font Awesome**: Icons for better visual communication
- **Responsive Images**: Optimized for different screen sizes

## 📱 Responsive Design

The page is fully responsive with breakpoints at:
- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px
- **Small Mobile**: Below 480px
